# Assets Directory

This directory contains templates and assets used by the P6 Project Controls skill.

## Report Templates

Future report templates will be stored here:

- Excel templates for variance reports
- HTML templates for schedule dashboards
- PDF templates for executive summaries
- PowerPoint templates for project reviews

## Sample Schedules

Sample XER files for testing and development:

- Simple test schedules
- Complex mining project examples
- Edge case schedules for validation testing

## Usage

Assets in this directory are referenced by the skill and used in generated outputs. They are not loaded into the context window but copied or modified as needed during execution.
